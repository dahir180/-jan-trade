# Jan Trade PWA

تطبيق سجل وتحليل صفقات قابل للتثبيت ويعمل دون إنترنت بعد أول زيارة.

## التشغيل الصحيح

ملفات PWA لا تُفتح بالنقر على `index.html` داخل تطبيق الملفات. يجب رفع المجلد إلى استضافة HTTPS مثل Netlify أو Cloudflare Pages أو GitHub Pages.

بعد النشر على iPhone:

1. افتح الرابط في Safari.
2. اضغط زر المشاركة.
3. اختر **إضافة إلى الشاشة الرئيسية**.
4. افتح Jan Trade من الأيقونة الجديدة.

## استيراد CSV

يجب أن يحتوي الصف الأول على الأعمدة التالية على الأقل:

`date,symbol,riskAmount,resultR`

الأعمدة الاختيارية:

`direction,session,fees`

مثال:

```csv
date,symbol,direction,session,riskAmount,resultR,fees
2026-07-14T08:30,XAUUSD,long,نيويورك,50,2,3
```

## الخصوصية

البيانات والصور محفوظة محليًا داخل متصفح الجهاز. استخدم **الإعدادات ← تنزيل نسخة كاملة** بصورة منتظمة.
